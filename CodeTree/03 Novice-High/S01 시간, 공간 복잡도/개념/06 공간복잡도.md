# 공간복잡도

- 메모리는 한정되어있으므로 더 적게 메모리를 차지할 수록 좋다.
- 공간복잡도 역시 시간복잡도처럼 점근적 표기법을 사용

```java
function example(n)
  set arr = [n][n]
  for i = 0 ... i < n
    for j = 0 ... j < n
      arr[i][j] = 1
```

- 다른 코드는 메모리를 차지 않기 때문에,
  - `set arr = [n][n]` 에서 `O(N^2)


## 메모리 제한

- 메모리는 C++ 기준으로 생각
  - `int`: 4 byte
  - `double`: 8 byte
  - `char`: 1 byte
  - `short`: 2 byte

### int 로 2천만이 80MB

```java
int a[2천만] : 80MB
int a[2백만] : 80 / 10 = 8MB
char a[2천만] : 80 / 4 = 20MB
double a[2천만] : 80 * 2 = 160MB
```

- int 로 2천만이 80MB 인 것을 이용하여 다른 타입도 어느정도 메모리를 사용하는지 유추가능
